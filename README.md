# TeachSite — Online Course Platform

TeachSite is an online learning platform that allows users to browse courses, enroll, manage their accounts, and access educational articles.

## Features

- Browse courses and view detailed information
- User registration and login
- Account management
- Access and read educational articles

## Getting Started

### Clone the repository

```bash
git clone https://github.com/ZahraAlimohammadi33/TeachSite---Online-Course-Platform.git
cd TeachSite---Online-Course-Platform
```

### Install dependencies
```bash
npm install
```

### Run the development server
```bash
npm run dev
```
